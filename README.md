# 원티드 프리온보딩 인턴십 사전 과제

## commit convention

| Tag Name | Description      |
| -------- | ---------------- |
| feat     | 새로운 기능 추가 |
| format   | 코드 포매팅 변경 |
| design   | UI 스타일 변경   |
| refactor | 코드 리팩토링    |
| fix      | 에러, 버그 수정  |
| remove   | 삭제             |
| docs     | 문서수정         |
| chore    | 기타 수정사항    |
